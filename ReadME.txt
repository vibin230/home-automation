This project is done with Three.Js and plain HTML and CSS.

To get started:


-----Download Three.JS zip file from threejs.org-----
-----Once downloaded extract the zip file-----
-----Copy the three.js and three.min.js file from build folder-----
-----Paste both the copied files on the same folder as HTML file-----
-----Inside the HTML add the location of both the pasted files to the "src" attribute of script tag-----
-----Save and open the HTML file on the browser-----


Project go through:

--> You can turn left and right using the left and right arrow keys in the keyboard.
--> There are setting and live report buttons on the navBav.
--> Click on settings button to open the Toggle menu for light and TV.
--> Click on live report button to open the live report container.
